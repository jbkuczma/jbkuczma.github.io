Featured Projects

Source code and others can be viewed on my Github.

NBAreact

A NBA mobile application created with React Native, allowing for the creation of an iOS and Android version to be created simultaneously. Features include current games being played along with team and player stats for that game, league standings, and team information such as roster, player stats, and current league standings for team stats.

rnStocks

A basic Stocks information mobile application created with React Native. Information that is provided include how the market is currently doing, information regarding an individual stock, and the ability to search for any stock or company.

Sounci

A nonintrusive SoundCloud desktop client built with web technologies. Tools used include ReactJS and Electron, along with HTML, CSS, and JavaScript.

TermScores

Easily check your favorite sport scores for from the command line. You can look up scores from a previous date or see who is playing later that day.

SoundCloud-dl

Download almost any song on SoundCloud via API calls. This was developed entirely for educational purposes and you should always support the artist.
