RhymeBot

The final homework for my data structures course (deemed as a mini project) involved creating a program in C++ using any data structure of our choosing and was capable of running various functions efficiently. When given a word, the RhymeBot would then return all words that rhymed or had an alliteration with that word. The user could then request specific parameters such as find a word that had so many syllables, had a certain number of rhyming syllables, had an alliteration with a different word, and was a certain part of speech.

Javascript Interpreter

Another of my homework assignments (deemed as a mini project) for data structures was to create a simplified javascript interpreter written in C++. The interpreter currently can handle writing to a .txt file, defining and assigning a double value via infix to postfix and then evaluating the equation, storing the definition of a function for later use, and then evaluating the definition of a function when called upon. Still in development are handling parameters and arguments passed into a function and handling if, if else, else statements.

Python Scripts

Since I already know Java and C++, over my winter break I decided to try and teach myself Python. These little programs are ones that I wrote while learning Python. Generate Passwords creates strong, random passwords based upon user requirements. A user can provide how many passwords they would like generated and the length of those passwords. An iOS app is currently in development. Weather provides the current and 5 Day forecast for the user based upon a provided city or zip code. Imgur Downloader scrapes subreddits on Reddit that the user provides and downloads any image that is hosted on imgur.com and meets the minimum score requirement.

Information for Bitcoin

This program was written in Java and monitors bitcoin information such as the high and low trades of the day, along with the most recent trade value. Information will then be written to a .txt file along with a time stamp for easy retrieval of past trades.
