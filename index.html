Scrabble

During my Spring 2015 semester at UB I took the second half of Intro to Computer Science for Majors. A big component of this class was a team project that lasted just under three months. All teams were assigned the task of creating a Scrabble-like game with Othello scoring. We used Git to manage the syncing of our code and learned the hard way to never be working on the same piece of code as someone else at the same time. This was a great learning expereience to end my Freshman year.

Guess the Number

A game where the user tries guessing a number that the computer is thinking of. This game was written in Java and the range of numbers is 1-100. There are two versions of the game: one that is played with the console in an IDE and another that is played with a GUI.

Tic Tac Toe

Everyone's favorite time-killing game that I wrote when I began learning C++. The game is played from a terminal window. I hope to add an AI component so someone can play against the computer.

This website
